# WorkTime Sync

**WorkTime Sync** — веб-приложение для автоматической актуализации рабочего времени сотрудников. Система анализирует данные из календарей, таск-трекеров и HR-систем, выявляет устаревшие или противоречивые графики, рассчитывает интегральный риск неактуальности и помогает планировать командную работу с учётом реальной доступности.

## Основные возможности

- **Управление профилями сотрудников**  
  Личные данные, рабочие графики (дни, часы, часовой пояс, формат работы), временные исключения (отпуск, больничный, командировка).

- **Карта доступности команды**  
  Тепловая карта доступности по дням и часам с отображением сотрудников, выпадающих из общего окна.

- **Анализ рисков и конфликтов**  
  Расчёт интегрального риска, доли встреч вне рабочего времени, уровня загрузки, конфликтов с HR-данными и календарём. Индивидуальные рекомендации для каждого сотрудника.

- **Планирование встреч**  
  Подбор оптимального времени на основе доступности участников, загрузки, отпусков и часовых поясов. Создание события в Google Calendar.

- **Подтверждение актуальности графика**  
  Кнопка обновления даты последнего редактирования графика.

- **Дашборд для руководителя и HR**  
  Сводка по актуальности, приоритет актуализации, распределение нагрузки, последние события.

## Технологический стек

| Компонент       | Технологии                                                                 |
|----------------|----------------------------------------------------------------------------|
| Фронтенд       | React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui                       |
| Роутинг        | React Router v6                                                           |
| Таблицы        | TanStack Table (DataTable)                                                |
| Графики        | Recharts                                                                  |
| Календарь      | React Day Picker, date-fns                                                |
| API клиент     | Swagger‑generated (OpenAPI) + Axios/fetch                                 |
| Бэкенд         | Python 3.14, FastAPI, SQLAlchemy, Alembic, PostgreSQL                  |
| Авторизация    | JWT (access + refresh)                                                    |

## Запуск проекта

```bash
git clone https://github.com/rerokai/wts.git
cd wts
npm install
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:5173`

Для полноценной работы нужно запусть backend (https://github.com/whoisblessed/WorkSync/)


## Роли пользователей

| Роль          | Описание                                                                 |
|---------------|--------------------------------------------------------------------------|
| **HR**        | Полный доступ: управление пользователями, командами, графиками всех сотрудников, просмотр всей аналитики. |
| **Руководитель** | Управление своей командой: просмотр графиков, создание встреч, анализ загрузки подчинённых. |
| **Сотрудник** | Только свой профиль, график, исключения, подтверждение актуальности.      |

## Основные модули

```
src/
├── api/                   # Сгенерированные клиенты OpenAPI (Api.ts, data-contracts.ts)

├── context/               # AuthContext (JWT, текущий пользователь)

├── pages/
│   ├── Panel/
│   │   ├── dashboard/     # Дашборд (сводки, графики, приоритеты)
│   │   ├── profile/       # Личный кабинет (данные, график, исключения)
│   │   ├── map/           # Карта доступности команды
│   │   ├── risk/          # Таблица рисков и конфликтов
│   │   ├── employee/      # Управление сотрудниками (только HR/руководитель)
│   │   └── meeting/       # Планирование встреч
│   └── Intro/             # Страницы входа / регистрации
└── AppRoute/              # Роутинг
```
